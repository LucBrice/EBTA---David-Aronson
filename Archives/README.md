# Archives

Ce dossier conserve les elements historiques qui ne font plus partie de la
surface active du protocole ou du runtime EBTA.

Regles :

- archiver plutot que supprimer quand l'historique peut expliquer une decision ;
- ne pas y deplacer une source active citee par `Protocole/` ou
  `Implementation/` ;
- journaliser les deplacements significatifs dans l'historique approprie ;
- garder `Protocole/Archives/` reserve aux archives documentaires du protocole.
